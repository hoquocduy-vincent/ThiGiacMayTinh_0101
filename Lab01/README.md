# README

## Giới thiệu chung

Repository này gồm 2 file Jupyter Notebook được viết dành chonngười mới bắt đầu học Computer Vision. Mục tiêu chính:

- Làm quen với việc xử lý ảnh bằng Python
- Hiểu sự khác nhau giữa hai thư viện phổ biến là PIL (Pillow) và penCV (cv2)
- Biết cách đọc, hiển thị và thao tác cơ bản với ảnh – nền tảng quan trọng trước khi học sâu hơn về AI, Machine Learning, và Deep Learning

---

## 1. File `2_1_1_Images_with_python_library_PIL.ipynb`

### Mục đích

Notebook này hướng dẫn cách sử dụng thư viện PIL (Pillow) – một thư viện đơn giản, dễ học, rất phù hợp cho người mới bắt đầu.

### Nội dung chính

Trong file này, bạn sẽ học được:

- Cách **cài đặt và import thư viện PIL (Pillow)**
- Cách **mở (load) một ảnh từ máy tính**
- Cách **hiển thị ảnh** trong Jupyter Notebook
- Truy cập các thông tin cơ bản của ảnh:
  - Kích thước (width, height)
  - Mode màu (RGB, L, …)
- Thực hiện một số thao tác cơ bản:
  - Resize ảnh
  - Crop (cắt) ảnh
  - Chuyển ảnh sang grayscale

## 2. File `2_1_2_Images_with_python_library_CV.ipynb`

### Mục đích

Notebook này giới thiệu thư viện OpenCV (cv2) – một thư viện rất mạnh, được sử dụng rộng rãi trong Computer Vision và AI thực tế.

### Nội dung chính

Trong file này, bạn sẽ học được:

- Cách **cài đặt và import OpenCV**
- Cách **đọc ảnh bằng OpenCV**
- Hiển thị ảnh bằng OpenCV và matplotlib
- Hiểu sự khác biệt giữa **BGR và RGB**
- Một số thao tác xử lý ảnh cơ bản:
  - Resize ảnh
  - Chuyển ảnh sang grayscale
  - Vẽ hình chữ nhật / đường thẳng lên ảnh
