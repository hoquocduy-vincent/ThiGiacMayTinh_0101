# README

## Giới thiệu chung

Repository này gồm 2 file ipynb dành cho người học **Computer Vision**.  
Mục tiêu chính của các bài học là:

- Làm quen với xử lý ảnh cơ bản bằng Python  
- Hiểu và sử dụng hai thư viện phổ biến: **PIL (Pillow)** và **OpenCV (cv2)**  
- Thực hiện các thao tác cơ bản với ảnh như: chuyển màu, lật ảnh, hiển thị ảnh  
- Chuẩn bị nền tảng cho các môn học nâng cao hơn như AI và Deep Learning  

---

## 1. File `2.2.1_basic_image_manipulation_PIL.ipynb`

### Mục đích

Notebook này giúp người học làm quen với thư viện **PIL (Pillow)** – một thư viện đơn giản, dễ sử dụng và rất phù hợp cho người mới bắt đầu.

### Nội dung chính

Trong file này sẽ học được:

- Cách **import thư viện PIL (Pillow)**
- Cách **mở (load) ảnh từ file**
- Cách **hiển thị ảnh** trong Jupyter Notebook
- Chuyển đổi ảnh sang định dạng **RGB**
- Thực hiện các thao tác xử lý ảnh cơ bản:
  - Lật ảnh theo chiều dọc (vertical flip)
  - Lật ảnh theo chiều ngang (horizontal mirror)
- Hiển thị và so sánh ảnh gốc với ảnh đã xử lý

---

## 2. File `2.2.2_basic_image_manipulation_open_CV.ipynb`

### Mục đích

Notebook này giới thiệu thư viện **OpenCV (cv2)** – thư viện mạnh mẽ và phổ biến trong lĩnh vực Computer Vision và AI.

### Nội dung chính

Trong file này sẽ học được:

- Cách **import thư viện OpenCV**
- Cách **đọc ảnh bằng OpenCV**
- Hiểu sự khác biệt giữa **BGR và RGB**
- Chuyển ảnh từ BGR sang RGB để hiển thị đúng màu
- Thực hiện các thao tác xử lý ảnh cơ bản:
  - Lật ảnh theo trục X (vertical flip)
  - Lật ảnh theo trục Y (horizontal flip)
- Hiển thị ảnh bằng **matplotlib**

---

## Kết luận

Hai file ipynb này giúp người học nắm được các khái niệm và thao tác **xử lý ảnh cơ bản** bằng Python.  
Đây là bước khởi đầu quan trọng trước khi tiếp cận các bài toán Computer Vision và AI nâng cao hơn.
